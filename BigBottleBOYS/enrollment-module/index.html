Κεντρικός Πίνακας Υποψηφίου




Από αυτή τη σελίδα μπορείτε να διαχειριστείτε την υποψηφιότητά σας. Επιλέξτε από το μενού για να ενημερώσετε τα προσωπικά σας στοιχεία, να συμπληρώσετε την αίτησή σας σταδιακά και να παρακολουθήσετε την πορεία της.


Στο My Profile μπορείτε να τροποποιήσετε στοιχεία όπως όνομα, τηλέφωνο και διεύθυνση. Το email παραμένει σταθερό, καθώς χρησιμοποιείται για την επικοινωνία και τις ενημερώσεις από το Πανεπιστήμιο.


Στο My Applications μπορείτε να επιλέξετε την αίτηση που θα συμπληρώσετε και να αποθηκεύετε την πρόοδό σας. Στο Application Status βλέπετε σε ποιο στάδιο βρίσκεται η αίτησή σας, ώστε να έχετε άμεση εικόνα της διαδικασίας.
